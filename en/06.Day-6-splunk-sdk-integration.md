In the PEAK Threat Hunting Framework, the AI acts as a "Security Analyst." An analyst cannot fabricate hypotheses out of thin air; they must base them on a real data baseline. By dynamically locking onto the data source and performing random sampling, we ensure the AI sees a different cross-section of the system state every time it patrols, preventing tunnel vision.

### 💻 Step 2: Inject SDK Sampling Logic in AOB

In the **Code Editor** on the right, we need to make two modifications.
**⚠️ Geek Iron Rule**: Comments and logs strictly use pure English.

**First: Import the SDK at the top of the file**
Scroll to the very first line of the code, and below the existing `import` list, add the Splunk SDK:

**Second: Rewrite the collect_events function**
Scroll down to locate `def collect_events(helper, ew):`. Completely replace the code beneath it with the following logic:

```python
def collect_events(helper, ew):
    """Implement your data collection logic here"""
    
    helper.log_info("PEAK AI Hunter started data collection cycle.")

    try:
        # 1. Acquire the session_key from the modular input context
        session_key = helper.context.meta.get("session_key")
        
        if not session_key:
            helper.log_error("Failed to acquire session_key from context.")
            return

        # 2. Connect to the Splunk service using the SDK
        service = client.Service(token=session_key)

        # 3. Dynamically hijack the user-selected Index
        target_index = helper.get_output_index()
        if not target_index:
            target_index = "main" # Fallback index for testing

        # 4. Construct the randomized baseline SPL
        # Geek Tip: We use `head 10000` BEFORE `random()` to prevent timeouts.
        search_query = (
            f"search index={target_index} "
            f"| head 10000 "
            f"| eval rand_val=random() "
            f"| sort 5 rand_val"
        )
        
        # 5. Use jobs.oneshot for fast, synchronous retrieval in JSON format
        kwargs_oneshot = {"output_mode": "json"}
        helper.log_info(f"Executing randomized oneshot SPL: {search_query}")
        
        oneshot_search_results = service.jobs.oneshot(search_query, **kwargs_oneshot)

        # 6. Parse the results
        reader = results.JSONResultsReader(oneshot_search_results)
        
        baseline_logs = []
        for result in reader:
            if isinstance(result, dict) and "_raw" in result:
                baseline_logs.append(result["_raw"])

        # 7. Log the extracted data length to verify success
        if len(baseline_logs) > 0:
            helper.log_info(f"Successfully fetched {len(baseline_logs)} random raw logs from index '{target_index}'.")
            helper.log_info(f"Sample Random Log [1]: {baseline_logs[0][:200]}...")
        else:
            helper.log_warning(f"SPL executed successfully, but returned 0 events from index '{target_index}'.")

    except Exception as e:
        helper.log_error(f"Execution error occurred during Splunk SDK search: {str(e)}")
```
